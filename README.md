# Full-Duplex-FD-Relay-System-Performance-Analysis-

– Developed a Full-Duplex Decode-and-Forward (FD-DF) relay communication simulator under Nakagami-m fading
channels using MATLAB.
– Analyzed BER and throughput performance versus SNR for QPSK, 16-QAM, and 32-QAM modulation schemes.
– Implemented path loss modeling, residual self-interference (RSI), AWGN noise, and Monte Carlo simulations for
realistic wireless channel analysis.
– Compared Full-Duplex and Half-Duplex relay architectures and validated simulation results using theoretical BER
analysis.
